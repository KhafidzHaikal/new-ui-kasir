📋 Implementation Examples

JavaScript Usage:

// Success Alert
showSuccessAlert('Berhasil!', 'Data telah disimpan');

// Warning Alert (Kadaluarsa)
showExpiredWarning(5, expiredProducts);

// Delete Confirmation
confirmDelete('Hapus Produk?', 'Data produk akan dihapus permanen')
  .then((result) => {
    if (result.isConfirmed) {
      // Lakukan penghapusan
    }
  });

// Toast Notification
showToast('success', 'Berhasil', 'Data telah diupdate', 3000);
