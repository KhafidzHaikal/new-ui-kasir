📱 Responsive Behavior

Desktop (>768px):

  • Badge position absolute di pojok kanan atas
  • Padding 8px 15px dengan border-radius 20px
  • Hover effect dengan transform dan shadow

Mobile (≤768px):

  • Badge tetap absolute tapi dengan ukuran lebih kecil
  • Padding 6px 12px dengan border-radius 15px
  • Font size dikurangi menjadi 10px

Small Mobile (≤480px):

  • Badge menjadi static dan inline
  • Padding 5px 10px dengan border-radius 12px
  • Font size 9px untuk readability
